# duplicate-master-prs-to-development

> A GitHub App built with [Probot](https://github.com/probot/probot) that When a PR to the master branch is created, create on also for development

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
